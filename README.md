# 🎮 Eclipse Frontier

## 📋 نبذة عن المشروع

**Eclipse Frontier** هي لعبة عالم مفتوح تم تطويرها باستخدام Godot 4.x. اللعبة توفر تجربة بناء قاعدة واستكشاف عالم مفتوح مع نظام موارد متقدم وأنظمة لعب متنوعة.

### المميزات الرئيسية

- ✅ **عالم مفتوح** قابل للاستكشاف
- ✅ **نظام بناء** متقدم مع 4 أنواع مباني
- ✅ **نظام موارد** (خشب، حجر، حديد، طعام)
- ✅ **نظام جرد** ديناميكي مع 30 فتحة
- ✅ **نظام حفظ** آمن مع Checksum والتشفير
- ✅ **نظام مهام** (رئيسية وجانبية)
- ✅ **نظام تطور اللاعب** (مستويات وخبرة)
- ✅ **نظام قتال** مع أعداء AI
- ✅ **واجهة مستخدم** احترافية
- ✅ **GitHub Actions** للبناء التلقائي
